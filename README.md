# Zero → One Builder

> A guided AI reasoning system that turns an uncertain idea into a concrete first step — not a chatbot, a structured 5-stage pipeline.

## What it does

Five stages of structured reasoning — each one builds on the last:

1. **Clarify** - surfaces gaps in your idea with targeted questions
2. **Analysis** - maps the problem, target user, value proposition, and resource fit
3. **Risk** - names the 4 assumptions your idea depends on, with honest confidence scores
4. **Perspectives** - genuinely disagreeing takes from a Founder, Investor, Engineer, and Customer
5. **Path** - 3 execution paths, scored, plus the smallest possible first experiment

The model shows its reasoning at every stage. It never picks for you — the decision stays with the human.

## Setup — from scratch in 5 steps

### 1. Get your Groq API key (free)

1. Go to [console.groq.com](https://console.groq.com)
2. Sign up for a free account
3. Navigate to **API Keys** → **Create API Key**
4. Copy the key — it starts with `gsk_`

---

### 2. Scaffold a new React project

Open your terminal and run:

```bash
npm create vite@latest zero-to-one-builder -- --template react
cd zero-to-one-builder
npm install
```

---

### 3. Add the project files

Download the two files from this repo and place them like this:

```
zero-to-one-builder/
├── src/
│   └── App.jsx        ← replace the default one with the downloaded App.jsx
├── .env               ← create this file in the root (see step 4)
└── ... (everything else Vite already created)
```

> Just overwrite the default `src/App.jsx` that Vite generated with the one from this repo.

---

### 4. Create your `.env` file

In the root of the project, create a file called `.env` and add your key:

```
VITE_GROQ_API_KEY=gsk_your_actual_key_here
```

Replace `gsk_your_actual_key_here` with the key you copied in step 1.

> ⚠️ Never share this file or commit it to GitHub — it contains your private key.

---

### 5. Run it

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) - you're good to go.
